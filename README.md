This repository documents my hands-on cybersecurity practice as part of the SOC Analyst learning path on [LetsDefend.io](https://letsdefend.io). It includes screenshots and brief notes from real-world simulations and Blue Team tasks to demonstrate progress and practical experience in threat detection and response.

- 📝 Personal notes and key takeaways from each exercise

- Log Analysis (Windows, Firewall, Web, Email)
